# Eksamen-Portfolie-Gregor-Pavlik

Eksame portfolie over 1.semester

Dette projekt er min personlige portfolio-hjemmeside, udviklet som en del af mit studie i Multimediedesign.  
Siden viser udvalgte projekter, mit designudtryk og mine færdigheder inden for HTML, CSS og JavaScript.

Formålet er at præsentere mit arbejde på en overskuelig måde.

English:
This project is my personal portfolio website, developed as part of my studies in Multimedia Design.
The site showcases selected projects, my design expression and my skills in HTML, CSS and JavaScript.
The purpose is to present my work in a clear way.
